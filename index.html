Web Development Skills

Learn HTML

HTML stands for Hyper Text Markup Language. HTML is the content of a web page. Examples of HTML include headings, paragraphs, images, links, lists, etc.

Check out the HTML Tutorial on w3schools.com.

Learn CSS

CSS stands for Cascading Style Sheets. CSS is a style of a web page. Examples of CSS include font size and color, borders, dimensions, layout, etc.

Check out the CSS Tutorial on w3schools.com.

Learn JavaScript

JavaScript is the programming language of the web. Java script helps to make web pages interactive and dynamic. Some common JavaScript tasks are:

  • add new HTML to page,
  • change the existing content / style,
  • react to user events (mouse clicks, key press. etc.)

Check out the JavaScript Tutorial on javascript.info.
